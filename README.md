# GameJam
An 8 hour game jam to develop a basic 2d game.

## How to Play
### Pre-requisites
* Python3
* pip3
* Install requirements while on master branch `$ pip3 install -r requirements.txt`
* [optional] make


### Versions
Current <br />
`$ git checkout master`

Game Jam 1 End State <br />
`$ git checkout GameJam1`

### Running Game
`$ make` <br />
or <br />
`$ python3 main.py`
